# Data folder

This folder contains the data files used to support the numerical simulations and figures in the manuscript.
The data are organized into six main datasets:
1. **planned_paths_planar**  
   Path planning results in the planar coordinate system for GA, PSO, GAPSO, and Improved-GAPSO, including both continuous path points and waypoint nodes.
2. **geo_projected_obstacles.mat**  
   Obstacle and region boundary data obtained by projecting real geographic coordinates into the planar coordinate system.
3. **geographic_regions.mat**  
   Real geographic region boundary data in longitude-latitude coordinates.
4. **curveData.mat**  
   Objective-cost evolution curves of different algorithms during the optimization process.
5. **route_geo.mat**  
   Geographic route generated by the Improved-GAPSO algorithm, including the start point, intermediate waypoints, and the goal point.
6. **all_circle_points.mat**  
   Real geographic coordinates of the threat regions, stored in longitude-latitude form. The data describe the circular threat-region boundaries used in the geographic map representation.
